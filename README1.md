# End-to-End Data Engineering Pipeline

## Overview
This project demonstrates a full data engineering workflow:
- Data ingestion from APIs and web scraping
- Data transformation and validation
- Data storage in SQLite
- Workflow orchestration using Airflow
- Data visualization with Streamlit

## Features
- Modular ETL architecture
- API + Web scraping ingestion
- Automated workflows (Airflow)
- Dashboard for data insights
- Dockerized environment

## Tech Stack
Python, Pandas, Airflow, SQLite, Streamlit, Docker
